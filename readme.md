# 📍 Meet-Me

**Meet-Me** est un projet personnel de recherche et développement.
Il vise à concevoir une application de geolocalisation entre 2 utilisateurs en temps reel.

L'objectif principal de ce projet est l'apprentissage de nouvelles stacks par la mise en pratique.

---

## 🚀 La Stack Technique

Le projet s'appuie sur les technologies suivantes pour garantir réactivité et scalabilité :

* **Framework :** [Next.js](https://nextjs.org/) (React)
* **Temps Réel :** WebSockets pour le suivi des positions.
* **Base de données :** NoSQL avec [MongoDB](https://www.mongodb.com/).
* **Distribution :** Déploiement en tant que **PWA** (Progressive Web App).
* **Infrastructure :** Conteneurisation complète via **Docker**.
* **Features Clés :** Système d'invitations par mail avec génération de **QR Codes**.

---

## 🛠️ Objectifs Pédagogiques

* Maîtriser le cycle de vie d'une application de la conception au déploiement.
* Implémenter une communication bidirectionnelle fluide.
* Gérer la persistance des données dans un environnement non-relationnel.
* Optimiser l'expérience utilisateur mobile via les standards PWA.
